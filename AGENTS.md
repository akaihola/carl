# Agent Instructions for Carl Project

## Browser Support & Modern Technologies

- **Target modern browsers only**: Firefox 145+, Chromium 143+ (November 2025 and later)
- Leverage cutting-edge CSS and browser APIs without legacy support concerns
- No need for polyfills or backward compatibility workarounds

## CSS-First Development

- Prioritize CSS for styling, layout, animations, and interactive states
- Use CSS Grid, Flexbox, CSS custom properties, and modern CSS features
- Implement hover states, focus states, and transitions purely in CSS
- Leverage CSS container queries, subgrid, and cascade layers where appropriate

- Prefer semantic HTML and semantic element use

- Write performant, accessible code without legacy constraints
